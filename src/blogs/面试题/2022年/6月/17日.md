## JSON.stringify()

- 使用 `JSON.stringify()` 方法来判断。

  <!-- more -->

  ```js
  if (JSON.stringify(obj) == '{}') {
    console.log('空对象')
  }
  ```



## Object.keys()

- 使用 `ES6` 新增的方法 `Object.keys()` 来判断
  ```js
  if (Object.keys(obj).length < 0) {
    console.log('空对象')
  }
  ```
