## 设置滚动条

```css
	overflow-x: scroll;
	white-space: nowarp;

```

## 隐藏滚动条

```css
	::-webkit-scrollbar {
		width: 0 !important
	}
	::-webkit-scrollbar {
		width: 0 !important
		height: 0
	}

```
