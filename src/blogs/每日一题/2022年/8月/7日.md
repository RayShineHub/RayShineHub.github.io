记录 `Vue` 常用指令及其用途。

<!-- more -->

|指令|描述|
|-|-|
| `v-for` | 循环数组，对象，字符串，数字。 |
| `v-on` | 绑定事件监听。 |
| `v-bind` | 动态绑定一个或者多个属性。 |
| `v-model` | 表单控件或者组件上创建双向绑定。 |
| `v-if`、`v-else`、`v-else-if` | 条件渲染。 |
| `v-show` | 根据表达式真假，切换元素的 `display`。 |
| `v-html` | 更新元素的 `innerhtml`。 |
| `v-text` | 更新元素的 `textcontent`。 |
| `v-pre` | 跳过这个元素和子元素的编译过程。 |
| `v-clock` | 这个指令保持在元素上直到关联实例结束编译。 |
| `v-once` | 只渲染一次。 |
