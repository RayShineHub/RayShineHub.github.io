# 🤚世人道我恋长安

远赴人间惊鸿宴，一睹大唐盛世颜。 总要去趟西安吧，吹吹明城墙上的风，走一走不夜城，去趟钟鼓楼，感受古城的繁华。

# 3、🥮 美食推荐

牛羊肉泡馍：肉多🥣浓，香醇美味，建议吃完再来一碗高汤。
肉夹馍：分3种，腊汁肉夹馍、老潼关肉夹馍、回民街肉夹馍。
biangbiang面：大碗宽面🥣。
甄糕：糯米和枣，不粘牙🦷。
水盆羊肉：水盆羊肉和白吉馍、锅盔是搭档，建议把馍泡在汤里。
葫芦鸡：西安名菜，号称长安第一味，建议辣椒🌶或者孜然。

⚠️ 不夜城好吃：

# 4、🚴 市内交通

  1. 地铁🚇：支付宝就可以刷，没有堵车风险，6:00 - 23:00，2元 - 8元。
  2. 公交🚌：旺季和上下班高峰经常堵车。
  3. 打车🚕：别坐黑车就行。
  4. 自驾🚗：高峰期一定要选好路线，不然经常堵2、3个小时。

⚠️ 能走高速尽量走高速，不然城里总是堵车。

# 5、🏠 市内住宿

住我那免费嘿嘿，倒是也有可能住景区旁边。

# 6、🔋 行李准备

证件：身份证、驾驶证、行驶证、学生证买票半价。
电器：单反、手机、充电器、充电宝、耳机、自拍杆等。
药品：肠胃💊、晕车💊、感冒💊
防晒：面膜、护肤品、🌂
穿搭：外套，早晚温差大；平底鞋/运动鞋、内衣裤。
其他：洗漱用品，床单，一次性马桶垫，如果住酒店尽量用自己的，尤其是毛巾。

# 7、⚠️ 注意事项

1、不要太累🥱，记得补水💦，开心就好❗️❗️❗️
2、大多景点都很大，要提前准备好零食。
3、提前订票的地方需要提前打问清楚。
4、❌不要买西安南站，远的很~
5、陕西历史博物馆是免费的门票，但是不能带自拍杆和三脚架，而且需要提前两三天预约。
6、陕西历史博物馆❌ 周一不开放。
7、秦始皇兵马俑下午五点就关闭了，建议三点半或者至少4点之前去。
8、🦵 不要错过大唐芙蓉园夜景🌃，除了表演，晚上灯亮了是最好看的。
9、西安城墙和碑林的票一起买便宜。
10、城墙白天没啥特色，晚上可以骑自行车。
11、回民街都是清真，本地人不吃～
12、提前准备点现金💰，碑林区很多地方收现金。
13、长恨歌表演时间据说是4月1日 - 10月31日。
14、钟楼附近的粉巷13号陕菜馆很好吃，可以去体验下～
15、高家大院是西安保存最完整的汉族居民院落之一，还有皮影戏
